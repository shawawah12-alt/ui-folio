# Food delivery home (React Native)

Home screen food delivery dengan location header, search bar SVG, kategori scrollable, banner promo, dan list restoran. Palet cream dengan accent terracotta.

## Preview

![preview](../../assets/previews/rn-food.png)

## Detail

- Background cream `#FFF8F0`
- Header lokasi dengan tombol 'Ubah'
- Search bar outline tipis
- Kategori horizontal scrollable
- Banner promo terracotta
- Card restoran dengan SVG icon + rating + waktu antar

## Cara pakai

```bash
cd react-native/food-delivery
npm install
npx expo start
```

## Customisasi

- Restoran: edit array `RESTOS` di `App.tsx`
- Kategori: edit array `CATS`
- Banner: ubah teks atau background color

## Tech stack

- React Native 0.74+
- Expo SDK 51+
- TypeScript

# 🇬🇧 English

Food delivery home screen with a location header, SVG search bar, scrollable categories, promo banner, and restaurant list. Cream palette with terracotta accent.

## Preview

![preview](../../assets/previews/rn-food.png)

## Detail

- Cream background `#FFF8F0`
- Location header with 'Change' button
- Thin-outline search bar
- Horizontal scrollable categories
- Terracotta promo banner
- Restaurant card with SVG icon + rating + delivery time

## How to use

```bash
cd react-native/food-delivery
npm install
npx expo start
```

## Customization

- Restaurants: edit the `RESTOS` array in `App.tsx`
- Categories: edit the `CATS` array
- Banner: change text or background color

## Tech stack

- React Native 0.74+
- Expo SDK 51+
- TypeScript
