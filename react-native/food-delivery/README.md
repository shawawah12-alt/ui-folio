# Food delivery home (React Native)

Home screen food delivery dengan location header, search bar, kategori scrollable, banner promo, dan list restoran. Palet cream dengan accent terracotta.

## Preview

![preview](../../assets/previews/rn-food.gif)

## Detail

- Background cream `#FFF8F0`
- Header lokasi dengan tombol "Ubah"
- Search bar outline tipis
- Kategori horizontal auto-rotate untuk demo
- Banner promo terracotta
- Card restoran dengan emoji + rating + waktu antar

## Cara pakai

```bash
cd react-native/food-delivery
npm install
npx expo start
```

## Customisasi

- Restoran: edit array `RESTOS` di `App.tsx`
- Kategori: edit array `CATS`
- Banner: ubah teks atau background color

## Tech stack

- React Native 0.74+
- Expo SDK 51+
- TypeScript
