# E-commerce product detail (React Native)

Product detail page dengan gallery 3 slide, rating stars, quantity selector, dan bottom bar (favorite + add to cart). Palet sand/cream dengan accent terracotta. Semua ikon SVG.

## Preview

![preview](../../assets/previews/rn-ecommerce.png)

## Detail

- Background cream `#FBF7F0`
- Gallery dengan 3 slide + dots indicator
- Quantity selector dengan tombol +/−
- Bottom bar fixed dengan tombol favorite outline + add to cart solid
- Tipografi Inter

## Cara pakai

```bash
cd react-native/e-commerce-product
npm install
npx expo start
```

## Customisasi

- Product data: ubah `SLIDES`, `name`, `price`, `desc` di `App.tsx`
- Quantity state: variabel `qty`
- Auto-rotate: ubah interval `2000` ms di `useEffect`

## Tech stack

- React Native 0.74+
- Expo SDK 51+
- TypeScript

# 🇬🇧 English

Product detail page with a 3-slide gallery, rating stars, quantity selector, and a bottom bar (favorite + add to cart). Sand/cream palette with terracotta accent. All icons are SVG.

## Preview

![preview](../../assets/previews/rn-ecommerce.png)

## Detail

- Cream background `#FBF7F0`
- Gallery with 3 slides + dots indicator
- Quantity selector with +/− buttons
- Fixed bottom bar with outline favorite + solid add-to-cart buttons

## How to use

```bash
cd react-native/e-commerce-product
npm install
npx expo start
```

## Customization

- Product data: change `SLIDES`, `name`, `price`, `desc` in `App.tsx`
