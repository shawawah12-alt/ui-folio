# 🔐 Login Screen Glassmorphism (React Native)

> Login screen dengan efek glassmorphism modern — blur background, card transparan, dan accent gradient.

## Fitur / Features

- Glassmorphism effect pakai `expo-blur`
- Dark mode ready
- Validasi form real-time
- Animasi smooth pakai `react-native-reanimated`
- Berjalan di iOS, Android, dan Web

## Cara Pakai / How to Use

```bash
cd react-native/login-screen-glassmorphism
npm install
npx expo start
```

## Tech Stack

- React Native 0.74+
- Expo SDK 51+
- TypeScript
- expo-blur, react-native-reanimated
