# Email inbox (Jetpack Compose)

Email inbox dengan palet cream paper `#F5F0E6`. Top bar minimalis, folder chips horizontal, list email dengan avatar berwarna, subject bold untuk unread, dan FAB 'Tulis' dengan ikon pencil SVG.

## Preview

![preview](../../assets/previews/compose-email.png)

## Detail

- Background cream `#F5F0E6`
- Card putih dengan divider tipis
- Avatar bulat dengan warna gradasi tan
- Unread dot golden `#D4A437`
- Folder chips dengan active state solid tan
- Tipografi Roboto

## Cara pakai

```bash
cd jetpack-compose/email-inbox
# Buka di Android Studio, atau:
./gradlew assembleDebug
./gradlew installDebug
```

## Customisasi

- Folder: edit list `folders` di `InboxScreen`
- Email data: edit list `emails` di `InboxScreen`
- Avatar warna: edit `AvatarBg1/2/3` di atas file

## Tech stack

- Jetpack Compose 1.6+
- Kotlin 1.9+
- Min SDK 24, Target SDK 34

# 🇬🇧 English

Email inbox with a cream-paper palette `#F5F0E6`. Minimalist top bar, horizontal folder chips, email list with colored avatars, bold subject for unread, and a 'Compose' FAB with a SVG pencil icon.

## Preview

![preview](../../assets/previews/compose-email.png)

## Detail

- Cream background `#F5F0E6`
- White cards with thin dividers
- Circular avatar with tan gradient colors
- Golden unread dot `#D4A437`
- Folder chips with solid tan active state
- Roboto typography

## How to use

```bash
cd jetpack-compose/email-inbox
# Buka di Android Studio, atau:
./gradlew assembleDebug
./gradlew installDebug
```

## Customization

- Folders: edit the `folders` list in `InboxScreen`
- Email data: edit the `emails` list in `InboxScreen`
- Avatar colors: edit `AvatarBg1/2/3` at the top of the file

## Tech stack

- Jetpack Compose 1.6+
- Kotlin 1.9+
- Min SDK 24, Target SDK 34
