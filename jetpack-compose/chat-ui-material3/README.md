# 💬 Chat UI Material 3 (Jetpack Compose)

> Chat UI dengan bubble pesan modern, attachment preview, typing indicator, dan input bar mengambang ala Telegram/WhatsApp.

## Fitur / Features

- Chat bubble dengan corner-cut khas Material 3
- Typing indicator animasi
- Input bar dengan attach button + emoji toggle
- Dark mode ready
- LazyColumn untuk performa tinggi
- Pesan terkirim & diterima tampil berbeda

## Cara Pakai / How to Use

```bash
cd jetpack-compose/chat-ui-material3
# Buka di Android Studio
./gradlew assembleDebug
# Atau install langsung:
./gradlew installDebug
```

## Tech Stack

- Jetpack Compose 1.6+
- Material 3
- Kotlin 1.9+
- Min SDK 24, Target SDK 34
