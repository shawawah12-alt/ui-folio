# News reader (SwiftUI)

News reader ala koran editorial. Brand "The Daily." dengan titik merah, horizontal tab strip, hero article dengan foto placeholder, dan list items dengan thumbnail. Tipografi serif untuk headline, sans untuk chrome.

## Preview

![preview](../../assets/previews/swiftui-news.gif)

## Detail

- Background off-white `#FAFAFA`
- Accent red `#C8102E`
- Brand serif dengan titik merah
- Tabs horizontal dengan underline merah
- Hero dengan foto gradient placeholder
- List item dengan thumbnail kanan

## Cara pakai

```bash
cd swiftui/news-reader
open NewsReader.xcodeproj
# Cmd+R di simulator
```

## Customisasi

- Headline: ubah teks di VStack hero
- Tab list: edit array `tabs`
- Article list: tambah/edit `NewsListItem`

## Tech stack

- SwiftUI 5
- iOS 17+
- Xcode 15+
