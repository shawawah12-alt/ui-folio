# Wallet — cards (SwiftUI)

Wallet app dark green forest `#0F1F1A`. Animated balance counter on load, balance card, kartu bank dengan chip emas, dan list transaksi dengan SVG icon berwarna. Accent green `#4ADE80`.

## Preview

![preview](../../assets/previews/swiftui-wallet.png)

## Detail

- Background forest dark `#0F1F1A`
- Card `#1A3329` dengan border `#1F3D2E`
- Balance counter animasi naik dengan `withAnimation`
- Chip kartu gradient emas
- Tx row dengan icon box berwarna SVG

## Cara pakai

```bash
cd swiftui/wallet-cards
open WalletCards.xcodeproj
# Cmd+R di simulator
```

## Customisasi

- Target balance: ubah `targetBalance`
- Kartu: ganti brand name, masked number, expiry
- Transaksi: tambah/edit `TxRow` di bawah

## Tech stack

- SwiftUI 5
- iOS 17+
- Xcode 15+

# 🇬🇧 English

Wallet app in dark forest green `#0F1F1A`. Animated balance counter on load, balance card, bank card with gold chip, and a transaction list with colored SVG icons. Green accent `#4ADE80`.

## Preview

![preview](../../assets/previews/swiftui-wallet.png)

## Detail

- Forest dark background `#0F1F1A`
- Card `#1A3329` with `#1F3D2E` border
- Balance counter animates up with `withAnimation`
- Gold gradient card chip
- Tx row with colored SVG icon box

## How to use

```bash
cd swiftui/wallet-cards
open WalletCards.xcodeproj
# Cmd+R di simulator
```

## Customization

- Target balance: change `targetBalance`
- Card: change brand name, masked number, expiry
- Transactions: add/edit `TxRow` below

## Tech stack

- SwiftUI 5
- iOS 17+
- Xcode 15+
