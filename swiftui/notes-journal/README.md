# Notes — journal (SwiftUI)

Editor jurnal dengan palet kertas cream `#F5EFE0`. Tipografi serif (Georgia/SF Pro) untuk body, sans untuk chrome. Bottom FAB dengan ikon pencil SVG.

## Preview

![preview](../../assets/previews/swiftui-notes.png)

## Detail

- Background paper `#F5EFE0`
- Accent tan `#8B6F47`
- Body serif dengan line-spacing lebar
- FAB lingkaran tan dengan ikon pencil SVG
- Top bar dengan back / title / done

## Cara pakai

```bash
cd swiftui/notes-journal
open NotesJournal.xcodeproj
# Cmd+R di simulator
```

## Customisasi

- Teks body: ganti string di `Text(...)` berturut-turut
- Warna paper: ubah `Color(red: 0.96, green: 0.94, blue: 0.88)`
- FAB action: ganti closure `action: {}`

## Tech stack

- SwiftUI 5
- iOS 17+
- Xcode 15+

# 🇬🇧 English

Journal editor with a paper-cream palette `#F5EFE0`. Serif typography (Georgia/SF Pro) for body, sans for chrome. Bottom FAB with a SVG pencil icon.

## Preview

![preview](../../assets/previews/swiftui-notes.png)

## Detail

- Paper background `#F5EFE0`
- Tan accent `#8B6F47`
- Serif body with generous line-spacing
- Tan circular FAB with SVG pencil icon
- Top bar with back / title / done

## How to use

```bash
cd swiftui/notes-journal
open NotesJournal.xcodeproj
# Cmd+R di simulator
```

## Customization

- Body text: replace the strings in the `Text(...)` calls
- Paper color: change `Color(red: 0.96, green: 0.94, blue: 0.88)`
- FAB action: replace the `action: {}` closure

## Tech stack

- SwiftUI 5
- iOS 17+
- Xcode 15+
