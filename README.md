## ✨ Tentang Repo Ini

Repo ini lahir dari satu masalah sederhana yang dihadapi hampir semua developer mobile: **waktu habis terbuang cuma untuk bikin screen pertama yang bagus**. Kamu sudah punya ide aplikasi keren, tapi begitu buka editor, bingung mau mulai dari mana. Onboarding gimana? Login-nya kayak gimana? Dashboard-nya layout apa? Nah, repo ini jawabannya.

Di sini kamu akan menemukan **lebih dari 100 desain UI mobile** yang sudah diimplementasikan di empat framework terpopuler: **Flutter, React Native, SwiftUI, dan Jetpack Compose**. Setiap UI kit dirancang dengan prinsip *pixel-perfect*, *responsive*, dan *accessible*. Artinya, kamu tidak cuma dapat screenshot cantik, tapi juga kode yang benar-benar berjalan dan siap di-deploy.

Repo ini cocok untuk:
- **Developer pemula** yang sedang belajar struktur UI mobile modern
- **Developer profesional** yang butuh boilerplate cepat untuk project baru
- **UI/UX Designer** yang ingin melihat implementasi nyata dari desain mereka
- **Startup founder** yang ingin cepat prototipe MVP tanpa nge-design dari nol
- **Pengajar / mentor coding** yang butuh referensi materi konkret

## 📥 Cara Pakai

### Opsi 1: Clone Seluruh Repo
```bash
git clone https://github.com/shawawah12-alt/awesome-mobile-ui-kit.git
cd awesome-mobile-ui-kit
```

### Opsi 2: Download UI Kit Tertentu Saja
Karena repo ini cukup besar, kamu bisa download folder tertentu menggunakan [DownGit](https://downgit.github.io/) atau `svn export`:
```bash
svn export https://github.com/shawawah12-alt/awesome-mobile-ui-kit/trunk/flutter/login-screen-minimalis
```

### Opsi 3: Pakai sebagai Template
Klik tombol **"Use this template"** di kanan atas halaman repo untuk membuat fork bersih sebagai starting point project kamu.

### Menjalankan UI Kit
Setiap folder UI kit punya `README.md` sendiri dengan instruksi spesifik. Contoh untuk Flutter:
```bash
cd flutter/login-screen-minimalis
flutter pub get
flutter run
```

## ✨ About This Repo

This repo was born out of one simple problem that nearly every mobile developer faces: **time wasted just building a good-looking first screen**. You already have a brilliant app idea, but the moment you open your editor, you're stuck wondering where to start. What should the onboarding look like? What about the login flow? What layout should the dashboard use? This repo answers all of those questions in one place.

Here you'll find **over 100 mobile UI designs** implemented across four of the most popular frameworks: **Flutter, React Native, SwiftUI, and Jetpack Compose**. Every UI kit is designed with the principles of *pixel-perfect*, *responsive*, and *accessible* in mind. That means you don't just get pretty screenshots — you get real working code that's ready to ship to production.

This repo is perfect for:
- **Beginner developers** learning modern mobile UI patterns
- **Professional developers** who need a fast boilerplate for a new project
- **UI/UX designers** who want to see real-world implementations of their designs
- **Startup founders** who want to quickly prototype an MVP without designing from scratch
- **Coding educators / mentors** looking for concrete teaching material

## 📥 How to Use

### Option 1: Clone the Whole Repo
```bash
git clone https://github.com/shawawah12-alt/awesome-mobile-ui-kit.git
cd awesome-mobile-ui-kit
```

### Option 2: Download a Single UI Kit Only
Since this repo is fairly large, you can grab a specific folder using [DownGit](https://downgit.github.io/) or `svn export`:
```bash
svn export https://github.com/shawawah12-alt/awesome-mobile-ui-kit/trunk/flutter/login-screen-minimalist
```

### Option 3: Use as a Template
Click the **"Use this template"** button at the top right of the repo page to create a clean fork as your project's starting point.

### Running a UI Kit
Each UI kit folder has its own `README.md` with specific instructions. Example for Flutter:
```bash
cd flutter/login-screen-minimalist
flutter pub get
flutter run
```
